Galton Board Explorer

This page turns the classic Galton board into a clean classroom demo for the Central Limit Theorem. As balls fall through successive left-or-right choices, the histogram builds a binomial distribution and begins to resemble the normal curve.

Galton Board — Binomial → Normal distribution
Rows: 8 → 9 bins
Spawn: 6/s
Speed: Med

How to use it

Start with Auto turned on and let the board run for a while. Then change the number of rows and watch how the shape of the histogram changes.

The gold outlines show the exact binomial model, while the dashed white curve shows the normal approximation.

Classroom idea: Ask students what random variable is being counted here: the number of right moves, or equivalently the final bin number after a fixed number of trials.
  • Increase rows to create more bins and a smoother bell-shaped pattern.
  • Use +100 quick or +1000 quick to build the histogram fast.
  • Turn Theory off first, ask for predictions, then turn it back on.
