For 4D, stereographic mode is the usual tesseract formula: (X,Y,Z)=2R(x,y,z)/(R-w). For dimensions above 4, the code uses one pole-perspective step (X,Y,Z)=2R(x₁,x₂,x₃)/(R-xₙ) after rotation in n-space. The extra dimensions still affect the picture through rotation, but the projection no longer iterates through several singularities.
