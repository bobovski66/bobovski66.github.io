# Christopher Lee-Jenkins

I'm a mathematician and educator working at the seams of differential topology (symplectic and contact), information geometry, machine learning, and creative computation. I build visual, hands-on tools both to teach and to test ideas. My current research program is **Observational Geometry**: the geometry of inference.
