# Christopher Lee-Jenkins

## About Me
I’m a mathematician and educator exploring the seams between differential topology (symplectic/contact), information geometry, machine learning, and creative computation. I like building visual, hands-on tools—both to teach and to test ideas. Current interests include discrete/continuous analogies (replicator flows, softmax bundles), interference patterns, NLSE/solitons, voltage/graph models, and topology-driven data analysis.
